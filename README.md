# KARMA Launchpad Frontend

## Setup

- clone the repo: git clone git@github.com:TradersClubDev/karma-frontend.git
- cp sample.env .env
- `yarn install` in the cloned folder
- `yarn dev` to start local dev server
- `yarn build` to build the project
